astroll IT Automation Online Installation

This installation manual enables you to easily install astroll IT Automation in your environment.

astroll can be installed on any operating systems that are compatible with Red Hat Enterprise Linux, and an Internet connection is required. ANSIBLE is selected as the automation software.

After installation is complete, by using the Quick Start guide below you can immediately experience astroll's powerful interface!

astroll IT Automation Quick start

This a quick start document to allow you to quickly familiarize yourself with the user interface of astroll IT Automation. astroll controls multiple northbound automation softwares, such as ANSIBLE.
In this quick start document, ANSIBLE is selected as the automation software because of its broad scope.
